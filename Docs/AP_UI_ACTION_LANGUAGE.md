# AP UI Action Language Floor (M-016)

Contract version: `ui-action-language-1.0.0`

## Scope lock
- Bottom-center action belt is always visible in runtime shell.
- Current verb strip must be readable at all times.
- Hover truth must reveal target/action context without swallowing the game view.
- Build quick row must be visible and reachable.
- Build access parity must exist through both key (`B`) and button lane.

## Out of scope
- No radar/minimap lane yet.
- No full build catalogue drawer implementation yet.
